Ping Pong AR

This project is a ping pong based game that I developed in Unity using Unity's AR Foundation and framework for iOS ARKit. All of the scripts for the game are written in C#. The game follows player versus AI logic for games with a scaling level of difficulty for the AI based on how many points the player scores. I implemented AI behavior by tracking movement of the ball and its difficulty level is based on how fast it keeps up with the ball.

  • Unity
  • ARKit
  • C#
  • https://github.com/david-tyler/david-tyler.github.io

Tower Stacking AR

Developed a tower stacking game app in Unity based on Unity's AR Foundation and framework for iOS, using C# to write scripts. Uses generated surfaces based on surface recognition and touch sensors recognized from tapping on a phones screen to spawn blocks. The blocks are also implemented with features applying physics to them as they can collide with one or fall off a surface.

  • Unity
  • ARKit
  • C#
  • https://github.com/david-tyler/david-tyler.github.io
